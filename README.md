# FINGERZ BOT

Hi and welcome, this is an implementation with Typescript of a Discord Bot for Web 3, based on Taquito for Tezos Ecosystem.

To test it.

```
git clone https://gitlab.com/Mache/fingerz-bot.git
cd fingerz-bot
yarn install
touch .env
TOKEN=""
CLIENT=""
yarn deploy
yarn start
```
